This is the about page.

The site you are visiting is a simple app built with Node.js and JavaScript.

The node server serves HTML, CSS and JavaScript files for requests made by the browser.


You can add books to your library on the Home page.

Books added are saved in local storage and will persist after the session ends.

They are not accessible on a different browser or device.


For persisting data, you can create a user account to store information in a database.

Data will be retrieved from the database and saved to indexedDB.

Once loaded, your library will be updated from data residing in indexedDB.


To create a user account in a mySQL database click here.


Your session is tracked in session storage.

Once your session is over, you will be logged out and indexedDB will be cleared.
